# 🍊 Catch the Orange

**Catch the Orange** is a simple arcade-style game built using Python and Pygame. The player controls a paddle at the bottom of the screen to catch falling oranges while avoiding misses.

---

## 🎮 How to Play
- Move your mouse to control the paddle
- Catch the falling oranges to earn points
- Each missed orange reduces your lives
- The game gets faster over time, increasing difficulty

---

## 🧠 Features
- Mouse-controlled paddle movement
- Random falling objects
- Score system
- Lives system
- Increasing difficulty as time progresses

---

## 🛠️ Technologies Used
- Python
- Pygame
